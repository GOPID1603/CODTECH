# CODTECH Artificial Intelligence Internship Tasks

This repository contains the completion of the four required Artificial Intelligence tasks for the CODTECH internship.

## Tasks Overview

### Task 1: Text Summarization Tool
A python script `task1_text_summarization.py` that utilizes HuggingFace's transformers library to summarize lengthy articles dynamically.

### Task 2: Speech Recognition System
A script `task2_speech_recognition.py` leveraging the `SpeechRecognition` library to transcribe sample audio files.

### Task 3: Neural Style Transfer
`task3_neural_style_transfer.py` uses pre-trained models from TensorFlow Hub to apply artistic styles to photographs.

### Task 4: Generative Text Model
`task4_text_generation.py` showcases text generation given a prompt using GPT models from HuggingFace Transformers.

## Setup Instructions

1. Clone the repository.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run any file using python:
   ```bash
   python task1_text_summarization.py
   ```
