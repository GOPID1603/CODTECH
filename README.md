## 🚀 Tasks Overview

### 📝 Task 1: Text Summarization Tool
**`task1_text_summarization.py`**
A script leveraging state-of-the-art Natural Language Processing. It utilizes **Hugging Face's `transformers`** library to ingest lengthy articles and dynamically generate concise, highly accurate summaries using the `distilbart-cnn-12-6` architecture.

### 🎙️ Task 2: Speech Recognition System
**`task2_speech_recognition.py`**
A speech-to-text transcription engine. By leveraging the **SpeechRecognition** library backed by Google Web Speech APIs, this script loads audio `.wav` files and converts spoken words into accurate plain text.

### 🎨 Task 3: Neural Style Transfer
**`task3_neural_style_transfer.py`**
A Computer Vision application that turns standard photographs into beautiful works of art. Utilizing pre-trained Magenta models from **TensorFlow Hub**, it merges the structural content of a base photo with the artistic style of another image (e.g., Van Gogh's Starry Night).

### 🤖 Task 4: Generative Text Model
**`task4_text_generation.py`**
A stunning showcase of Large Language Models (LLMs). This script utilizes the famous **GPT-2 Model** to contextually understand a provided prompt and creatively generate continuing text, completing the sentence artificially.

## 🛠️ Setup Instructions

### 1️⃣ Clone the Repository
Clone this exact environment to your local machine:
```bash
git clone https://github.com/GOPID1603/CODTECH.git
cd CODTECH
```

### 2️⃣ Install the Dependencies
It is highly recommended to create a virtual environment first. Once inside your environment, fetch all necessary dependencies via:
```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Tasks
You can run any script individually to see it in action. (Note: Media files like `content.jpg`, `style.jpg`, and `sample_audio.wav` are already included out-of-the-box!)

```bash
python task1_text_summarization.py
python task2_speech_recognition.py
python task3_neural_style_transfer.py
python task4_text_generation.py
```
